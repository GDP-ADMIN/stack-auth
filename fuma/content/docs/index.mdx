# Stack Auth Documentation

Welcome to the Stack Auth documentation. Stack Auth is a managed user authentication solution. It is developer-friendly and fully open-source (licensed under MIT and AGPL).

## Getting Started

Choose your framework to get started:

- [Next.js](/docs/next/getting-started/setup)
- [React](/docs/react/getting-started/setup)
- [JavaScript](/docs/js/getting-started/setup)
- [Python](/docs/python/getting-started/setup)

## Features

Stack Auth provides a comprehensive set of features for authentication and user management:

- Pre-built authentication components
- User dashboard
- Account settings
- Multi-tenancy & teams
- Role-based access control
- OAuth connections
- Passkeys
- Impersonation
- Webhooks
- Automatic emails
- User session & JWT handling
- M2M authentication
